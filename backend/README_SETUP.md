# Setup Demo Data - دليل إعداد البيانات التجريبية

## نظرة عامة

هذا السكريبت (`setup_demo.py`) يقوم بإنشاء بيانات تجريبية شاملة للمشروع، تتضمن:

- ✅ 5 وحدات دراسية كاملة
- ✅ 36 دورة دراسية مع أفضل 3 فيديوهات يوتيوب لكل دورة (108 فيديو)
- ✅ TDs وامتحانات لكل وحدة
- ✅ موارد تعليمية
- ✅ بيانات تقدم ودرجات واقعية
- ✅ مستخدم تجريبي جاهز

## المتطلبات

- Python 3.9+
- MongoDB Atlas connection (أو MongoDB محلي)
- متغيرات البيئة المطلوبة في `.env`:
  ```
  MONGODB_URL=mongodb+srv://...
  MONGODB_DB_NAME=student_ai
  ```

## التشغيل

### الطريقة 1: من مجلد المشروع الرئيسي
```bash
cd backend
python setup_demo.py
```

### الطريقة 2: من أي مكان
```bash
python backend/setup_demo.py
```

## استكشاف الأخطاء

### خطأ الاتصال بـ MongoDB
إذا رأيت خطأ SSL أو Connection Error:
1. تحقق من اتصال الإنترنت
2. تحقق من صحة `MONGODB_URL` في `.env`
3. جرب إعادة التشغيل بعد بضع دقائق

### خطأ في الاستيراد
```bash
pip install -r requirements.txt
```
